Project Overview
This project investigates the relationship between U.S. 
crude oil prices and refined fuel prices from 1986 to 2025. 
The analysis was conducted as part of a data analytics capstone
 for Western Governors University (D502).

The objective was to determine whether fluctuations in crude oil prices
significantly impact refined fuel markets using regression modeling 
and lagged correlation analysis.

Tools and Technologies:
Python (pandas, scikit-learn, matplotlib)
Jupyter Notebook
Tableau Public
Microsoft Excel (for initial data inspection)
